# CheatSheets

#### `JavaScript / ReactJS / TypeScript`

# [https://cheatsheets-js.com/](https://cheatsheets-js.com/)

## Getting Started

```bash
yarn install

yarn dev

yarn build

yarn deploy
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

## Stack

- Next.js
- TypeScript
- Styled-Components

Also use:

- Scss
- dotenv
- gtag
- ftp-deploy
